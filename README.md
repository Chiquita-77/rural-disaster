# rural-disaster
乡村灾害预警与防灾指导小程序（毕业设计）

## 项目简介
本项目是面向乡镇应急管理场景开发的防灾减灾类微信小程序，集**灾害信息上报、实时预警推送、防灾科普指导、数据统计分析、地图定位、便民通知**于一体，解决基层防灾信息不及时、流程不规范、数据不统一等问题。系统采用前后端分离架构，支持多端运行，界面简洁易用，可直接部署于乡镇政务场景使用，具备完整的实战价值与毕业设计亮点。

## 技术栈
- **前端**：uni-app + Vue3 + JavaScript + 微信小程序 API
- **后端**：Java + Spring Boot + MyBatis
- **数据库**：MySQL
- **其他**：Postman（接口测试）、Git（版本控制）

## 核心功能
- **灾害上报**：用户可快速提交灾害位置、类型、描述及现场图片
- **预警查看**：实时展示官方发布的灾害预警信息，支持按地区筛选
- **防灾科普**：提供各类灾害的预防、自救、互救知识文章与视频
- **数据统计**：可视化展示区域内灾害类型、频次、处理进度等数据
- **地图定位**：集成地图功能，快速定位灾害地点与周边避难场所
- **便民通知**：发布乡镇防灾通知、应急演练安排等便民信息
